#Steps to run the app

    1. clone the repo
    2. npm install
    3. npm run dev
    4. (Optional) create a database in MySQL to create users when you submit the form

#Packages that i used

    Client side
    1. formik
    2. yup
    3. axios
    4. antd
    
    Server side
    1. express
    2. mysql2
    3. cors
